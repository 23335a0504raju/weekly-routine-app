# Weekly Routine Planner 📅

A stunning, interactive, and completely mobile-responsive web application designed to help you organize your week, manage tasks efficiently, and maximize productivity. With a visually rich and intuitive interface, planning your week has never been easier.

## 🌟 Features

- **Responsive Mobile-First Design**: The application intelligently transforms its layout based on the screen size. On mobile, it switches from a grid view to a beautifully stacked card layout for optimal readability and touch interaction.
- **Glassmorphism UI**: A premium user interface with subtle transparency, beautiful gradients, and sleek animations that provides a modern user experience.
- **Interactive Schedule**: Click anywhere to type tasks. Easily add time slots or breaks with dedicated action buttons.
- **Smart Column Copying**: Quickly copy an entire day's schedule to another day, or to the entire week, to save time setting up repeating routines.
- **Undo / Redo Support**: Made a mistake? Seamlessly step back or forward through your recent edits.
- **Print & Export Support**: Print your schedule directly to a perfectly formatted PDF, or export and import your schedule data as a JSON file to transfer between devices.
- **Shareable Links**: Generate a shareable URL containing your entire schedule to easily send to colleagues or open on other devices.
- **Auto-Save**: Changes are automatically saved locally, ensuring you never lose your progress.

## 🚀 Getting Started

Getting started with the Weekly Routine Planner is incredibly simple. Since it is entirely front-end driven, there are no complex dependencies or backend setups required.

### Prerequisites
A modern web browser (Google Chrome, Mozilla Firefox, Apple Safari, or Microsoft Edge).

### Installation & Usage
1. Clone this repository or download the source code:
   ```bash
   git clone https://github.com/yourusername/weekly-routine-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd weekly-routine-app
   ```
3. Open `index.html` in your web browser. That's it!

## 🎨 UI Showcase

The app uses CSS Variables (Custom Properties) to provide a centralized and easily customizable theme. It relies on beautiful typographic scales powered by Google Fonts (Inter and Outfit).

*   **Desktop View**: Displays a comprehensive week-at-a-glance grid with sticky headers and inline actions.
*   **Mobile View**: Each time block becomes an easy-to-read vertical card. Days are clearly labeled without needing a bulky horizontal scrollbar, providing a truly native app feel.

## 🛠️ Technology Stack
- **HTML5**: Semantic structure.
- **CSS3**: Vanilla CSS utilizing flexbox, grid, and media queries for responsive design.
- **JavaScript (ES6)**: Vanilla JS for state management, history stacking, and local storage persistence.
- **Feather Icons**: Beautiful, minimalist open-source icons.
